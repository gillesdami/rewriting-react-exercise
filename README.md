# Rewriting react

This repository offers a step by step tutorial on how to rewrite some react functionalities to better understand what they do behind the scene.

It does not aim to reproduce any react optimization.

You can preview the app we will build with our custom react [here](https://no2ehi.github.io/simple-todo-app-react/).
![simple-todo-app-react](https://user-images.githubusercontent.com/35344951/214790785-e90ddd84-8d3d-482f-baaa-30b4ce6032d2.jpg)

## Requirements

nodejs and yarn.

```sh
yarn # install the deps
yarn start # run a dev server
```
